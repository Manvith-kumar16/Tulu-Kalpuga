## ğŸ’¡ Overview

Tulu-Kalpuga is an interactive web platform designed to teach the Tulu language, a classical Dravidian language spoken in the coastal districts of Karnataka, India.  
The project uses a clean, modern frontend stack to deliver a smooth learning experience focused on alphabets, pronunciation, vocabulary and writing practice.  
The goal is to preserve and promote Tulu through accessible and engaging digital tools.

---

## ğŸ¯ Key Features

* Complete coverage of Tulu alphabets including Vowels and Consonants
* Image based examples for each letter to support intuitive learning
* Dedicated writing practice page with canvas for freehand tracing
* Numbers learning module with verified and accurate visuals
* Responsive UI that works well on all screen sizes
* Modular project structure for easy expansion
* JSON based local data storage
